# Электронный каталог книг

Курсовой проект на Laravel: каталог книг с онлайн-бронированием, продлением срока и админ-панелью.

## Возможности

- регистрация и вход на базе Laravel Breeze;
- каталог книг на Blade;
- поиск по названию, автору, жанру и ISBN;
- бронирование доступной книги;
- личный кабинет с продлением и отменой бронирований;
- админ-панель;
- CRUD книг, авторов и жанров;
- просмотр всех бронирований и смена статуса;
- управление общим и доступным количеством экземпляров.

## Установка

```bash
composer install
npm install
php artisan key:generate
php artisan migrate:fresh
php artisan optimize:clear
```

Для демонстрационных данных можно дополнительно выполнить:

```bash
php artisan db:seed
```

После `db:seed` доступен администратор:

- email: `admin@library.local`
- пароль: `password`

Если сидер не запускать, первый зарегистрированный пользователь автоматически получит роль администратора.
